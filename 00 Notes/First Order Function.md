## 2️⃣ First-Order Function

### Definition

> A **first-order function** is a function that **does not accept another function as an argument and does not return a function**.

It operates only on **data values**.

### Example

```js
function add(a, b) {
  return a + b;
}
```

* Takes numbers
* Returns a number
* No function input/output

### Key Point

First-order functions are **not special** — they are simply **normal functions**.
