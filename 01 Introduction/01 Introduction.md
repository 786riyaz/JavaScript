# JavaScript – Introduction

## What is JavaScript?

**JavaScript** is a programming language used to make websites **interactive** and **dynamic**.  
It works alongside:

- **HTML** – Structure  
- **CSS** – Styling  

JavaScript adds **behavior** to web pages.

> JavaScript is popularly known as the **"Language of the Web"**.

---

## History of JavaScript

- **1995**  
  Brendan Eich created JavaScript in just **10 days** for Netscape to make web pages interactive.

- **1996**  
  Introduced in **Netscape Navigator**.  
  Initially called **LiveScript**, later renamed to **JavaScript**.

- **1997**  
  JavaScript was standardized as **ECMAScript (ES1)** to ensure consistency across browsers.

- **2009**  
  **ES5** introduced important features such as:
  - Strict mode
  - New array methods

- **2015**  
  **ES6 (ECMAScript 2015)** brought major improvements:
  - Arrow functions
  - Classes
  - Modules
  - Let & const

- **Today**  
  JavaScript is used everywhere:
  - **Frontend** (websites)
  - **Backend** (servers using Node.js)

---

## Where Does JavaScript Run?

### In the Browser
- Executed by built-in JavaScript engines  
  - Example: **V8 engine** in Google Chrome
- Works in browsers such as:
  - Chrome
  - Firefox
  - Safari
  - Edge

### On Servers
- Using **Node.js**, JavaScript can run on servers
- Enables **full-stack development** with a single language

---

## What Can JavaScript Do?

- Update content dynamically  
  *(e.g. live news feeds)*

- Respond to user actions  
  *(clicks, typing, scrolling)*

- Perform animations  
  *(slideshows, fading effects)*

- Communicate with servers  
  *(load/save data using APIs)*

- Build full applications:
  - Web applications
  - Mobile applications
  - Desktop applications

---

## Fun Facts About JavaScript

- JavaScript and Java are **not related**, despite the similar name
- One of the **most popular programming languages globally**
- Runs on **over 98% of websites** today
- Google’s **V8 engine** made JavaScript significantly faster and more efficient
