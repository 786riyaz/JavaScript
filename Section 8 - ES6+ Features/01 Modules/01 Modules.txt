Modules

The standard way to organize code into reusable files using `import` and `export`.
  * **`export`:** Makes variables, functions, or classes available for use in other files.
  * **`import`:** Brings those exported members into the current file.

// --- file: math.js ---
export const PI = 3.14;
export function add(a, b) { return a + b; }

// --- file: app.js ---
import { PI, add } from './math.js'; 
console.log(add(PI, 1));
